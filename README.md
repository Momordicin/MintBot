<h2 id="english">English</h2>

A local-first AI character companion for your desktop. Fully customizable personas, voice dialogue, desktop overlay, and long-term memory — all data stays on your machine.

> This project is under active development, currently at Phase 1.

### Features

**Conversation**
- Real-time chat with your AI character via SSE streaming
- Multi-turn reply support with natural pacing
- Complete message history stored locally

**Desktop Overlay**
- Auto-switches to a floating overlay when the chat window is minimized
- Smart behavior based on active window: always-on-top / hide / move to another display
- Character artwork changes based on emotion tags (GIF and static image support)

**Voice**
- Voice input via faster-whisper ASR
- Voice replies via GPT-SoVITS v2 (streaming, plays from first sentence)
- Emotion-guided voice tone

**Memory System**
- Dual-track memory: recent context injected directly, historical context via RAG recall
- BGE-large vector embedding with local sqlite-vec index
- Auto summarization and entity aggregation
- Bidirectional emotion engine: character emotion + user emotion perception

**Human-Computer Interaction**
- Local system operations: launch apps, adjust volume, take screenshots, etc.
- MCP extension interface (reserved)
- Proactive dialogue scheduler: time-based, event-based, emotion threshold triggers

**Mobile**
- Access via Cloudflare Tunnel
- Shared session with desktop, real-time sync

### Tech Stack

| Layer | Technology |
|---|---|
| Desktop | Electron + React |
| Core Service | Node.js + Fastify (PM2) |
| Model Providers | Anthropic / OpenAI API / Ollama |
| Local Model | Ollama (optional, Qwen3 / ChatGLM etc.) |
| ASR | faster-whisper (Python HTTP service) |
| TTS | GPT-SoVITS v2 (Python HTTP service) |
| Embedding | BGE-large (Python HTTP service) |
| Database | SQLite + sqlite-vec |
| Win32 FFI | koffi |
| Config Hot-reload | chokidar |

Supports both external APIs (Anthropic / OpenAI) and local Ollama — configured via `config.json`. No cloud storage dependency.

### Quick Start

> Full setup docs will be published when Phase 1 is complete.

**Prerequisites**
- Node.js 20+
- pnpm 8+
- Python 3.10+
- Ollama (optional) **or** an Anthropic / OpenAI API key

```bash
# Clone
git clone https://github.com/Momordicin/MintBot.git
cd MintBot

# Install dependencies
pnpm install

# Copy config files
cp .env.example .env
cp config.example.json config.json
# Edit config.json — add your API key or configure local Ollama

# Seed the database (insert a test preset)
pnpm seed

# Start development
pnpm dev
```

### Character Configuration

Place character assets under `assets/characters/<name>/`:

```
assets/characters/my-character/
  ├── manifest.json     # emotion label → artwork file mapping
  ├── idle.gif
  ├── happy.gif
  └── ...
```

`manifest.json` example:

```json
{
  "name": "my-character",
  "format": "gif",
  "emotions": {
    "idle":    { "gif": "idle.gif",    "png": "idle.png" },
    "happy":   { "gif": "happy.gif",   "png": "happy.png" },
    "curious": { "gif": "curious.gif", "png": "curious.png" },
    "sleep":   { "gif": "sleep.gif",   "png": "sleep.png" }
  },
  "fallback": "idle"
}
```

### Privacy

- All conversation data is stored in local SQLite — nothing is uploaded
- When using external APIs (Anthropic / OpenAI), conversation content is sent to the respective provider; privacy-conscious users should configure local Ollama
- Sensitive fields (message content, API keys) support AES-256-GCM field-level encryption
- Local deployment keys are managed by the system keychain (Windows Credential Manager), never written to disk in plaintext

<h2 id="chinese">中文</h2>

一个运行在本地的 AI 角色伴侣桌面应用。支持自定义角色设定，语音对话，桌面悬浮窗，以及长期记忆。数据完全留在本机，隐私优先。

> 项目正在开发中，当前处于 Phase 1。

### 功能特性

**对话**
- 与自定义 AI 角色实时对话，SSE 流式输出
- 支持多段回复，模拟真实对话节奏
- 消息历史完整留存

**桌面悬浮窗**
- 聊天窗口最小化后自动切换至桌面悬浮窗，后台常驻
- 根据活跃窗口自动置顶 / 隐藏 / 跳屏
- 角色立绘根据情绪标签实时切换（支持 GIF 动图和静态图片）

**语音**
- 语音输入（faster-whisper ASR）
- 语音回复（GPT-SoVITS v2，流式合成，首句即播）
- 情绪标签指导语音语调

**记忆系统**
- 双轨记忆：近期对话直接注入 context，历史对话 RAG 召回
- BGE-large 向量 embedding，sqlite-vec 本地索引
- 自动摘要压缩，实体聚合
- 双向情绪引擎：角色情绪 + 用户情绪感知

**人机交互**
- 本地系统操作：启动应用、调整音量、截图等
- MCP 扩展接口（预留）
- 主动对话调度器：定时、事件、情绪阈值触发

**手机端**
- 通过 Cloudflare Tunnel 访问本地服务
- 与桌面端共享会话，实时同步

### 技术栈

| 层级 | 技术选型 |
|---|---|
| 桌面框架 | Electron + React |
| 核心服务 | Node.js + Fastify（PM2 守护） |
| 模型调用 | @anthropic-ai/sdk / OpenAI API / Ollama |
| 本地模型 | Ollama（可选，Qwen3 / ChatGLM 等） |
| ASR | faster-whisper（Python HTTP 服务） |
| TTS | GPT-SoVITS v2（Python HTTP 服务） |
| Embedding | BGE-large（Python HTTP 服务） |
| 数据库 | SQLite + sqlite-vec |
| Win32 FFI | koffi |
| 配置热生效 | chokidar |

模型调用支持外部 API（Anthropic / OpenAI）和本地 Ollama，由配置文件决定。数据完全留在本机，不依赖任何云端存储。
